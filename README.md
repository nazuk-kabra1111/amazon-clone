Amazon Clone (Frontend)
A visual representation of the Amazon homepage built using HTML and CSS, focusing on layout design, responsiveness, and UI structuring.

Features
1.Fully designed Amazon-style homepage
2.Responsive navigation bar
3.Product grid layout
4.Hero section with banner
5.Footer similar to Amazon
6.Clean and structured UI

Tech Stack
- HTML5
- CSS3 (Flexbox & basic styling)

What I Learned:
-> Structuring a real-world UI layout
-> Using Flexbox for alignment
-> Managing spacing, margins, and positioning
-> Improving design consistency
